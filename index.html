HTML & CSS Roadmap

1. HTML Document Structure

Learned about <!DOCTYPE html>, <html>, <head>, <body>. Practiced building the skeleton of a webpage. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

2. Text, Links & Lists

Discovered elements like <p>, <a>, <ul><li> and made my first list of favorite topics.

  • Semantic paragraph structures
  • Clickable links [MDN Docs]
  • Organized lists for notes

3. CSS Selectors & Styling

Wrote my first CSS to change colors, font, spacing, and add modern effects! 

body {
  background: linear-gradient(120deg, #dbeafe, #fff);
  color: #1e1e2f;
  font-family: 'Inter', Arial, sans-serif;
}
h1 {
  color: #2563eb;
}
